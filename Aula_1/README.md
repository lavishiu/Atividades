# Aula 01 — HTML básico

## Objetivo da aula

Construir uma página HTML mínima usando estrutura base (doctype, head e body), tags semânticas (`header`, `main`, `footer`) e elementos fundamentais (`h1`, `p`, `a`, `img`).

## Capacidades (SENAI)

- CT7: Reconhecer e utilizar recursos básicos do HTML.
- CS3: Organizar arquivos do projeto e manter consistência.

## Pré-requisitos

- VS Code instalado
- Navegador (Chrome/Edge/Firefox)
- Noções de pastas/arquivos

## O que foi feito em sala

- Estrutura base do HTML.
- Cabeçalho com título e navegação.
- Seções no conteúdo principal.
- Imagem com `src`, `alt` e `width`.
