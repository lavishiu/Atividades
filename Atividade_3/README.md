# Atividade 03 — Leiautes com divs

## Objetivo

Reproduzir, com **HTML** e **CSS**, os leiautes propostos na lista de exercícios, utilizando principalmente `div`, `display: flex` e propriedades de espaçamento para construir as formas e organizar os blocos na página.

## Capacidades avaliadas

- CT7: Reconhecer e aplicar propriedades de CSS para composição de leiautes com `div`.
- CS3: Organizar o projeto, os arquivos e o processo de desenvolvimento.

## Critérios técnicos mínimos (checklist)

- [ ] Cada exercício foi montado com estrutura em HTML usando `div`.
- [ ] Os blocos foram organizados com CSS, especialmente `display: flex`, quando necessário.
- [ ] As formas foram construídas com propriedades de dimensão e espaçamento, conforme a proposta da atividade.
- [ ] Os exercícios estão centralizados e organizados visualmente na página.
- [ ] Os títulos dos exercícios estão destacados de forma consistente.
- [ ] O CSS está em arquivo separado.
- [ ] O código está limpo, legível e sem trechos desnecessários.

## Rubricas

### (CT7)

- Insuficiente: Não conseguiu reproduzir os leiautes propostos; uso incorreto ou muito limitado de `div` e CSS; blocos desorganizados, desalinhados ou diferentes da estrutura esperada.
- Básico: Reproduziu parcialmente os leiautes; há compreensão inicial do uso de `div` e `flex`, mas com inconsistências de alinhamento, espaçamento ou proporção.
- Adequado: Reproduziu corretamente os exercícios propostos; utilizou `div` e propriedades de CSS com coerência; blocos organizados, centralizados e visualmente próximos ao modelo.
- Avançado: Além do adequado, manteve alta consistência visual entre os exercícios, organizou muito bem o CSS, evitou repetição desnecessária e demonstrou cuidado estético na apresentação dos títulos e dos blocos.

### (CS3)

- Insuficiente: Projeto desorganizado; arquivos incompletos ou fora do padrão; dificuldade de localizar ou validar a entrega.
- Básico: Organização parcial; arquivos presentes, mas com pouca padronização ou revisão.
- Adequado: Estrutura clara, arquivos corretos e código bem separado entre HTML e CSS; entrega completa e fácil de validar.
- Avançado: Além do adequado, demonstra planejamento visual e técnico, padronização clara entre os exercícios e cuidado na apresentação final do arquivo.
